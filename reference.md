# Reference

## &lt;div>-like elements
- &lt;header>
- &lt;nav>
- &lt;main>
- &lt;article>
- &lt;section>
- &lt;aside>
- &lt;footer>
- &lt;summary>

## list types
- &lt;ul> - unordered list (uses bullets)
- &lt;ol> - ordered list (uses numbers)
    - &lt;li> - list item (item in the list)

## input types
- <form>
- <input>
- <button>
- <select>
    - <option>
